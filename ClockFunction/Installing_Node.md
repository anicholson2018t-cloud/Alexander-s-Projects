# ClockFunction

A simple employee clock-in and timesheet system built with JavaScript and Node.js.

## Features

- Clock In
- Clock Out
- Other entries with optional notes
- Automatic date and time logging
- Saves records to Timesheet.txt
- Anonymous user support

## Requirements

- Node.js installed

## How to Run

1. Download the project
2. Open a terminal in the project folder
3. Run:

```bash
node ClockFunction.js
```

4. Follow the instructions on screen

## Code Language

JavaScript (`.js` file)
