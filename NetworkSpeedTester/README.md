# 🌐 Network Speed Tester

A simple browser-based network speed tester built with **HTML, CSS, and JavaScript**.

## 📌 Features

✅ Download speed test  
✅ Ping test  
✅ Public IP detection  
✅ ISP information  
✅ Mobile-friendly design  
✅ Dark tech-style interface  
✅ Runs directly in a web browser  

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- Chart.js
- GitHub Pages

## 📂 Project Structure
NetworkSpeedTester/
│
├── index.html
├── style.css
├── script.js
└── README.md

## 🚀 How To Use

1. Download or clone this repository.
2. Open `index.html` in a browser.
3. Press **Start Test**.
4. View your network results.

## 🌍 GitHub Pages

This project can be hosted for free using GitHub Pages:

1. Go to:
Settings → Pages

2. Select:
Deploy from branch

3. Choose:
main branch

4. Save.

Your website will become available online.

## ⚠️ Notes

- Upload speed requires a server that accepts uploads, so it is not included in the GitHub Pages version.
- Results can vary depending on Wi-Fi signal, device, and network traffic.
